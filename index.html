AWS 解決方案架構師專業級 (SAP) 考試學習指南

本學習指南基於真實的 AWS SAP 考試題目，整理了通過考試所需的關鍵知識點。內容按主題分類，方便學習和複習。

目錄

  • 網絡架構 - VPC、Transit Gateway、Direct Connect、Route 53 等
  • 安全與身份 - IAM、加密、安全組、WAF 等
  • 計算服務 - EC2、Auto Scaling、ELB、容器服務等
  • 存儲服務 - S3、EBS、EFS、Storage Gateway 等
  • 數據庫服務 - RDS、DynamoDB、Aurora、Redshift 等
  • 無服務器架構 - Lambda、API Gateway、Step Functions 等
  • 遷移策略 - 遷移工具、策略和最佳實踐
  • 成本優化 - 成本控制、資源優化、預留實例等
  • 高可用性與災難恢復 - 多可用區、多區域、備份恢復策略等
  • 監控與運維 - CloudWatch、CloudTrail、Config 等

考試概述

AWS 解決方案架構師專業級 (Solutions Architect Professional, SAP) 考試是 AWS 認證體系中最高級別的架構師認證之一。

SAP 考試的特點：

  • 側重於複雜場景下的架構設計
  • 需要對 AWS 服務有深入理解
  • 強調成本優化、高可用性、安全性和性能效率
  • 考察多服務集成能力和最佳實踐應用
  • 包含遷移策略和混合架構設計

本學習指南根據真實考題提取關鍵知識點，按主題分類整理，幫助您系統性地準備 AWS SAP 考試。點擊上方目錄中的各個主題，深入學習相關內容。
